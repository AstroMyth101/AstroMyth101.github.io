## Design direction

- Static GitHub Pages site, no build pipeline required.
- Dark engineering-lab visual style with optional light mode.
- Mobile-responsive layout.
- Portfolio-first structure: projects, experience, skills, resume snapshot, and contact.
- Public website intentionally omits phone number.

## Featured project

The first full case study is the Soft Robotic Rehabilitation Glove Controller, presented as a bench-validation and research platform. It does not claim clinical approval, patient readiness, or human testing approval.
